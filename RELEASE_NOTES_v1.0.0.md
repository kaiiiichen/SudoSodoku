## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/kaiiiichen/SudoSodoku.git
   ```

2. Open in Xcode:

   ```bash
   open SudoSodoku.xcodeproj
   ```

3. Configure signing and build!

See [README.md](README.md) for detailed build instructions.
